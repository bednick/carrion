# Carrion

Рогалик-автобатлер о жизни после большой войны.

Недавно завершилось огромное сражение трёх сторон — магической, кавалерийской и бронированной. Стороны разошлись,
но поле битвы не пустует: там осталась нежить из поднятых павших бойцов, дикие животные, привлечённые запахом мяса,
и мародёры, пришедшие за ценным железом.

Игрок — один из тех, кто обосновался в лагере на краю поля и живёт вылазками за трофеями. Не герой и не спаситель
мира — выживальщик, который снаряжается, отправляется в экспедицию, сражается с обитателями каждой зоны и её
боссом, собирает лут и возвращается в лагерь за новой партией снаряжения.

Лор не объясняется напрямую — картина случившегося складывается через найденные предметы, встреченных мобов и
особенности каждой из зон.

Конечная цель — пробиться через все зоны по краям карты и сойтись в последней битве с финальным боссом в самом
центре поля, где встретились все три фракции.
