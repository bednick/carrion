# Контент: оружие (слот `hand_right`)

Источник правды по слоту `hand_right`: домен, ось, формы урона (архетипы), характеристики предмета и список
предметов слота. Общие правила системы предметов (R1–R9) и функциональность слотов —
[`content.items.md`](content.items.md); боевая идентичность фракций — [`factions.md`](factions.md);
событийная модель — [`combat-events.md`](combat-events.md).

## Домен слота

**Основной урон героя.** `hand_right` — **единственный слот прямо внутри треугольника фракций**
(в отличие от `legs`, который вне боя). Каждый предмет объявляет поток атаки (`attackInterval`) и авторит урон
на `attack_ready` (см. [`combat-events.md`](combat-events.md) §5).

Оружие — **чемпион двух фракций** и намеренно не третьей:

- **Звери** (много целей / напор) — ответ: **cleave / прошив** (один замах косит волну) и **мультихит**
  (много ударов гасят свору, попутно кормят вампиризм).
- **Мародёры** (одна броняцель) — ответ: **тяжёлый пробой** (редкие сильные удары, мало роллов блока) и
  **крит** (крупный удар обходит флэт-броню).
- **Нежить** (бой про время) — **не контрится оружием (канон A).** Анти-нежить — это **реновация**
  (вампиризм/реген), домен другого слота. Оружие лишь **кормит** её: `fast`/мультихит = больше событий
  `damage` = больше проков лайфстила (эмерджентно, [`combat-events.md`](combat-events.md) §6). Дать оружию
  собственный анти-нежить урон = схлопнуть треугольник, поэтому запрещено.

## Ось слота

Домен — урон; ось — **профиль подачи урона**: **концентрация** (одна цель, крупный удар) ↔ **распределение**
(много целей / много ударов). Полюса оси — «весь урон в одну точку» против «размазать по доске».

Поверх оси живут **дескрипторы-риддеры**, не образующие своих осей (R1 — одна ось на слот):

- **темп** — `slow` / `fast` (интервал атаки);
- **крит** — шанс крупного удара;
- **контроль** — стан (стаки на цели → порог);
- **накопление** — раскачка урона (ramp, стаки на цели).

Архетип = **идентичность** (форма урона: набор тегов для синергий и тултипов) + **один профильный стат** (R5),
который скейлит редкость. Разные архетипы одного слота ведут себя по-разному.

### Отклонённые направления

- **Пронзание брони (флэт-игнор)** — «мод плюс-на-минус», минимальное взаимодействие. Его роль (обход брони)
  закрывает **крит** — тот и броню обходит, и юзабелен вне Мародёров (просто менее оптимален).
- **Собственный анти-нежить урон** (оверкил-перенос / DoT сквозь фазы) — ломает треугольник. Канон A: нежить —
  через реновацию, не оружие.
- **Уклонение как инструмент оружия** — уклонение отдано **врагу-Нежити** (защита, бестелесность),
  см. [`factions.md`](factions.md). Оружию не принадлежит.
- **Анти-нежить как отдельная форма урона** — см. выше, повтор для явности: оружие покрывает 2 фракции из 3.

## Формы урона (архетипы)

Открытый веер (пополняется контентом). Столбец «движок» — что нужно для реализации; статус зависит от него.

| Архетип              | Полюс оси       | Профильный стат (R5)         | Фракция-цель       | Движок                                     |
|----------------------|-----------------|------------------------------|--------------------|--------------------------------------------|
| **Тяжёлое (burst)**  | концентрация    | урон (интервал фикс.)        | Мародёры           | есть (`standardWeapon`)                    |
| **Сбалансированное** | центр           | урон+темп без перекоса       | нейтр. (стартовое) | есть (`standardWeapon`)                    |
| **Лёгкое (fast)**    | к распределению | темп (интервал падает)       | кормит вампиризм   | есть (`standardWeapon`)                    |
| **Мультихит**        | распределение   | число ударов за взмах        | Звери / кормёжка   | есть (автор спавнит N `attack`)            |
| **Крит**             | концентрация    | шанс крита (множитель фикс.) | Мародёры           | лёгкое (RNG в авторе, свой `behavior.ts`)  |
| **Cleave / прошив**  | распределение   | урон по нескольким целям     | Звери              | ⏳ roadmap: multi-target attack + доска 2×2 |
| **Раскачка (ramp)**  | концентрация    | величина стака урона         | Мародёры           | ⏳ roadmap: система стаков (на цели)        |
| **Стан**             | контроль        | шанс / скорость накопления   | Звери / Мародёры   | ⏳ roadmap: статусы с длительностью + стаки |

> **Доска сейчас линейна** (1 ряд × 4). Cleave трактуется как **удар по линии**, но всё равно требует
> multi-target attack (движок целит одну ячейку). Прошив/рикошет и «удар по соседним» ждут доски **2×2** —
> см. [`roadmap.md`](roadmap.md). До этого сплеш-архетипы проектируются, но не шипятся.

## Характеристики предмета

Combat-статы живут в `behavior.ts` (R9), скейл — по редкости, `stats()` — единый источник правды для тултипа и
скупщика (R4). Оружие объявляет:

- **урон** (`damage`) — базовый удар; у большинства архетипов скейлит редкость.
- **интервал** (`attackInterval`, сек) — темп потока атаки; у `fast`/мультихита падает с редкостью.
- **профильный rider** (по архетипу, ровно один — R5):
    - крит: `crit_chance` (скейл) + `crit_mult` (фикс.);
    - мультихит: `hits` (число `attack` за взмах);
    - cleave: охват целей (по линии / соседние);
    - ramp: `ramp_step` + `ramp_cap` (прирост за удар по одной цели, сброс при смене цели);
    - стан: `stun_chance` **или** `stun_buildup` + порог, `stun_duration`.

**Лестница апгрейда** — как в семействе `legs`: `common → uncommon → rare → epic` (4 ступени), скейл капается на
`epic`; `legendary` — авторская редкость (значения вручную, скейлом недостижима). Скейл — `scaleByRarity(base, r, s)`
= `base·s^index` (`src/items/scaleByRarity.ts`).

Числа — стартовая калибровка, не финал; балансятся на реальных зонах. Все ключи в конфигах — `snake_case`.

### Древко — стартовое тяжёлое

Слабое тяжёлое (`shaft`): низкий урон, длинный интервал; урон растёт с редкостью, интервал фикс. Учебный «пробой»
для первого знакомства с Мародёрами.

### Обломок посоха — тяжёлый пробой

Основное тяжёлое (`broken_staff`): редкий сильный удар. Профиль — урон (скейл ×1.5), интервал фикс. Апекс
одноцелевого продавливания брони; мало роллов блока по определению (низкий темп).

### Гнутый клинок — сбалансированное

Базовая линия (`bent_blade`): средние урон и темп, без перекоса. Точка отсчёта и универсальный старт до сборки
профильного билда.

### Ржавый нож — лёгкое

Быстрое лёгкое (`rusty_knife`, dual-slot `hand_right`/`hand_left`): низкий урон, короткий интервал (падает с
редкостью). Двигатель под вампиризм (частые `damage` = частые проки лайфстила) — косвенный вклад в бой с нежитью.

### Парные клинки — мультихит

Автор спавнит **2** `attack` за один `attack_ready` (движок уже умеет, спецполя не надо — два честных `attack` →
два `damage` → лайфстил прокает дважды). Профиль — число ударов и темп. Чемпион против Зверей (быстро режет свору)
и сильнейший кормилец вампиризма. Пишется руками поверх `standardWeapon`.

### Клевец — крит

Тяжёлое с крит-rider: с шансом наносит крупный удар (`crit_mult` фикс. ×2.0), профиль — `crit_chance` (скейл ×1.35
по редкости). **Обходит флэт-броню** (крупный удар − флэт = всё равно много), поэтому анти-Мародёр; но **не обходит
блок/уклонение** (полное гашение по шансу) — частичная контра. Юзабелен и против прочих, просто менее оптимален.
Риск дисперсии гасит связка `heavy + crit` (редкий крупный удар и так в теме тяжёлого). Движок: RNG в авторе,
свой `behavior.ts`.

### Секира — cleave

Удар по нескольким целям (сейчас — **по линии**; на доске 2×2 — по ряду). Профиль — урон по каждой цели. Чемпион
против Зверей, R6-ворота по своре. **Требует multi-target attack** (движок целит одну ячейку) — см.
[`roadmap.md`](roadmap.md). До этого не шипается.

### Цеп — раскачка (ramp)

Урон растёт за серию ударов по **одной** цели (`ramp_step` за удар, кап `ramp_cap`), **сброс при смене цели**.
Наказывает вязкого одиночку — против своры не работает (цель всё время меняется), поэтому само-балансится под
Мародёров. Профиль — величина стака. **Требует системы стаков (на цели)** — см. [`roadmap.md`](roadmap.md).

### Булава — стан

Стан-шкала копится с ударов (`stun_buildup` → порог) **или** флэт-шанс за удар (`stun_chance`); на срабатывании
враг не действует `stun_duration` секунд (не капает свой поток атаки). Душит напор Зверей и ответку Мародёра
(стан = нет окна блока/контрудара); нежить не контрит (воскрешение цело — канон A). Профиль — шанс/скорость
накопления. **Ловушка баланса:** `fast` + стан-по-шансу = лок насмерть; рычаги — короткий стан, высокий порог,
затухающая отдача. **Босс — стан-резист** (иначе фазы ломаются). **Требует статусов с длительностью + стаков** —
см. [`roadmap.md`](roadmap.md).

## Судьба старых предметов (решено)

- `shaft`, `broken_staff` — **остаются** (стартовое / основное тяжёлое; сайдгрейд по ступени редкости).
- `bent_blade` — **остаётся** (сбалансированное, точка отсчёта).
- `rusty_knife` — **остаётся** (лёгкое; dual-slot сохраняем).
- Ни один hand_right-предмет на удаление не идёт (в отличие от `legs`, `head`, `body_lower`).

## Порядок реализации

1. **Крит** (`war_pick`) — движок лёгкий (RNG в авторе), сразу даёт анти-Мародёр форму помимо heavy.
2. **Мультихит** (`twin_blades`) — движок готов, почти бесплатно; закрывает кормёжку вампиризма и напор Зверей.
3. **Cleave** (`broadaxe`) — после multi-target attack; R6-ворота по Зверям.
4. **Ramp** (`flail`) — после системы стаков.
5. **Стан** (`mace`) — после статусов с длительностью (+ стаки).

Пункты 3–5 — за фундаментами из [`roadmap.md`](roadmap.md) (доска 2×2, система стаков, статусы).

## Открытые вопросы

1. **Число потоков атаки.** Одно оружие = один поток стамины. Мультихит — 2 `attack` на один взмах (не второй
   поток). Второе оружие (`hand_left`) — отдельный поток. Свести в UI полосок стамины при сборке билда.
2. **Cleave на линейном ряду vs 2×2.** До 2×2 «по линии» = удар по всем занятым ячейкам ряда. Определить,
   шипать ли cleave на линейной доске или ждать 2×2 целиком.
3. **Крит и порядок хуков.** Крит рождается в авторе (до `damage`); проверить, что флэт-броня врага
   (хук на `damage`) применяется к уже-криченному числу, а не наоборот.
4. **Дома предметов.** В лут-таблицы каких зон класть новое оружие: heavy/крит — к Мародёрам, cleave/мультихит —
   к Зверям (R6-покрытие пула).
5. **Скейл мультихита.** Профиль — число ударов (растёт с редкостью, как рероллы у `appraiser_boots`) или урон
   за удар при фикс. 2 ударах? Решить при реализации.
