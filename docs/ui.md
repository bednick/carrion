# Интерфейс и экраны

## Общее

- Разрешение: адаптивное — игра рендерится в логическом пространстве 16:10 и растягивается на весь экран игрока через
  Phaser Scale Manager (`FIT`). Внутренняя логика позиционирования работает в относительных координатах, не в абсолютных
  пикселях.
- Тёмная цветовая схема
- Управление мышью во всех сценах лагеря
- Хвер-эффекты на интерактивных объектах: подсветка + анимация + подпись

---

## Лагерь (CampScene)

Статичная 2D-сцена. Персонаж не перемещается — всё управляется мышью.

### Объекты на сцене

- Костёр (центр) + герой (Силач)
- Кузнец (слева)
- Скупщик (справа)
- Сундук / Стойки для брони (позиция TBD)

### Взаимодействие с объектами

- Наведение → объект подсвечивается, играет анимация интереса, появляется подпись
- Клик → открывается панель взаимодействия

### Панель кузнеца

Только улучшение (1 предмет + эссенция → следующая редкость) и разборка (1 предмет → эссенция).
Рецептов и комбинирования двух предметов нет.

```
┌──────────────────────────────────────────┐
│  [Кузнец]           │  [Сундук]          │
│  Предмет + эссенция │  [ ] [ ] [ ] ...   │
│  → след. редкость   │  Вкладки по типам  │
│  [Улучшить][Разобрать] (наведение = сколько│
│                     │   эссенции за разбор)│
└──────────────────────────────────────────┘
```

### Панель скупщика

Две вкладки:

**Вкладка «Магазин»** — покупка проходок в новые зоны за золото. Показывает все зоны, пропуск в которые можно купить (не разблокированы через квест). Те же зоны одновременно отображают кнопку [Купить Xg] прямо на карте — оба способа ведут к одному результату.

**Вкладка «Задания»** — список активных квестов (только активные, без истории).
Каждый квест: название + условие + прогресс + награда. Клик не нужен — просто список.

```
┌──────────────────────────────────────────┐
│  [Магазин]  [Задания]  │  [Сундук]       │
│                        │  [ ] [ ] ...    │
│  (активная вкладка)    │  (наведение=    │
│                        │   цена, клик=   │
│                        │   продать)      │
└──────────────────────────────────────────┘
```

### Панель сундука / стоек для брони

Единый интерфейс, открывается при клике на любой из объектов.

- Вкладки: Оружие / Броня / Аксессуары / Разное
- Сортировка: по редкости, по типу слота
- **Три стойки для брони** — переключатель пресетов (1 / 2 / 3): панель с силуэтом героя и 7 слотами (крест
  вокруг портрета: `ring`/`head`/`amulet` сверху, `hand_left`/`body`/`hand_right` в центре, `legs` снизу).
  Каждый пресет хранится отдельно; перед вылазкой игрок выбирает, с какой стойки идти

## Карта

Встроена в CampScene — отдельной сцены нет. Открывается в лагере (клик по костру / кнопка «На вылазку»).

### Элементы

- Визуальная карта с неравномерными областями
- У каждой области: название, статус (заблокирована / открыта / пройдена)
- Заблокированные: тёмные, иконка замка
- Заблокированные с доступным пропуском: иконка замка + кнопка **[Купить Xg]**
- Открытые: активные, клик → начать экспедицию
- Пройденные: отмечены (иконка черепа босса / галочка)
- Три стартовые зоны на разных краях карты
- Финальная зона в центре (открывается только после прохождения всех 9 зон)

**Зоны, контент которых ещё не реализован** (средние и финальная в MVP): отображаются как заблокированные с надписью **«В разработке»** вместо иконки замка. Клик недоступен.

### Взаимодействие

- Наведение на область → показывает превью: название, диапазон боёв, тип мобов
- Клик на открытую область → переход к экспедиции
- Клик [Купить Xg] → диалог подтверждения → `MetaStore.unlockArea()` (аналогично кнопке в «Магазине» скупщика)

## Экспедиция (ExpeditionScene / GameScene)

### Полоса прогресса (вверху)

```
[Лагерь] ──●──────●──────●──────●──────[БОСС]
             ✓      ✓      →      ○       ○
             1      2      3      4       5
```

- ● = столкновение (отметка)
- ✓ = пройдено
- → = текущее (анимация персонажа стоит здесь)
- ○ = предстоит
- Полоса плавно заполняется во время ходьбы между отметками

### Зоны экрана

Лутом в бою не управляют (собирается автоматически). Нижняя панель показывает экипировку героя и табы
стоек 1/2/3 — активную стойку можно сменить прямо в походе, в том числе в бою (снаряжение и герой
пересобираются на лету, HP переносится). Последний выбор запоминается и используется при следующем входе.

```
y=0..40   [Полоса прогресса экспедиции]              [Скорость ×1/×2/×4]
y=40..360 ┌──────────── Сражение 1280×320 (полная ширина) ──────────────┐
          │  Герой ← HP/ATK бары              Моб → HP/ATK бары         │
          │  (анимация атаки / ходьбы)                                  │
          └─────────────────────────────────────────────────────────────┘
y=344..396 ◄══════════════ Лента добычи (полная ширина, инфо) ════════►
y=396..800 ┌─Экипировка (только просмотр)─┐
           │  7 слотов текущего билда     │
           └──────────────────────────────┘
```

### Зона сражения

- Герой слева, моб справа
- HP бар (зелёный) и ATK бар (жёлтый/оранжевый) под каждым
- У боссов: несколько ATK баров (независимые атаки)
- Анимации: удар (красный тинт + откат), блок (золотой тинт + всплеск), числа урона

### Лента добычи

Горизонтальная полоса на всю ширину экрана. Показывает, что выпало с мобов, — **пассивно**, без
взаимодействия: весь лут уходит в сундук автоматически (рюкзака и ручного подбора нет).

**Жизненный цикл предмета:**

1. Моб умирает → предмет падает сверху вниз анимацией (с позиции моба до ленты)
2. Проезжает влево и **автоматически уходит в сундук** (флоатер `+предмет`); золото и эссенция —
   отдельными флоатерами (`+N G`, `+N эссенции`)

**Визуальные сигналы:**

- Предметы с редкостью выше common подсвечены рамкой своего цвета
- Hover → тултип с описанием предмета (просмотр, без действий)

### Награда за босса (выбор)

После победы над боссом зоны показывается **экран выбора награды**: N предметов из `boss.loot`, из которых
игрок берёт **один** (draft 1 из N) — гарант за прохождение зоны.

- Выбранный предмет уходит в сундук; остальные варианты не выдаются
- Кнопка [Забрать] → возврат в лагерь, зона отмечается пройденной
- В **центральной зоне** этого экрана нет: победа над финальным боссом ведёт сразу на экран завершения игры
  (см. ниже)

### Тултипы

- Предмет: редкость (цветом), допустимые слоты, стата из `describe()`, описание
- Герой: интервал атаки, текущие параметры
- Моб: атака, интервал, редкость

### Кнопки управления

- `×1 / ×2 / ×4` — скорость игры (правый верхний угол)
- Кнопки досрочного выхода нет: экспедиция идёт до боя с боссом или до гибели (механика побега удалена)

---

## Завершение игры

Финальная (центральная) зона — только бой с боссом, без рядовых боёв, ленты добычи и награды-выбора.
Победа над финальным боссом ведёт на **экран завершения игры**:

- Итоговый экран с исходом (по лору — поле битвы наконец покорено)
- Возврат в лагерь; далее — свободная игра (можно перепроходить зоны за лутом)

## Числа и эффекты (флоатеры)

| Тип     | Цвет    | Текст   | Длительность       |
|---------|---------|---------|--------------------|
| Урон    | Красный | `-X`    | 1.35s вверх + fade |
| Лечение | Зелёный | `+X HP` | 1.4s вверх + fade  |
| Эссенция | Голубой | `+X`    | 1.2s вверх + fade  |
| Золото  | Жёлтый  | `+X G`  | 1.2s вверх + fade  |
