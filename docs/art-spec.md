# Арт-спецификация

## Персонаж и шаблон кадра

В игре один играбельный персонаж — **Силач**. Кадр персонажа задаёт шаблон, к которому крепятся оверлеи
экипировки:

- Фиксированный размер canvas на кадр
- Фиксированные точки крепления экипировки (голова, обе руки, торс, ноги)
- Фиксированный набор анимаций с постоянным числом кадров

Оверлеи экипировки рисуются относительно этих точек и общие для всех предметов — **один комплект оверлеев
на персонажа**.

---

## Размеры спрайтов

| Тип                        | Размер       | Примечание                              |
|----------------------------|--------------|-----------------------------------------|
| Персонаж — лагерь (стоп-кадр) | **300×600 px** | статичный спрайт для CampScene        |
| Персонаж — бой (кадр анимации) | 64×96 px  | базовая модель, спрайт-листы           |
| Предмет-оверлей            | 64×96 px     | то же, что боевой кадр                 |
| Иконка предмета            | 40×40 px     | для инвентаря, SVG                     |
| Моб                        | TBD          | зависит от типа                        |
| Босс                       | TBD          | крупнее обычных мобов                  |

### Позиция персонажа в лагере

Спрайт 300×600 px отображается в натуральный размер. Позиция задаётся как **нижняя середина** (bottom-center):

| Персонаж | Bottom-center | Центр спрайта (в коде) |
|----------|---------------|------------------------|
| Силач    | (560, 600)    | (560, 300)             |

Тень рисуется эллипсом шириной `300 × 0.9 = 270 px`, высотой `20 px` на уровне нижней середины.

## Анимации персонажа

Персонаж имеет следующий набор анимаций (число кадров фиксировано).

| Анимация | Кадров | Описание                          |
|----------|--------|-----------------------------------|
| `idle`   | 6      | Лёгкое покачивание в стойке       |
| `walk`   | 8      | Ходьба (используется между боями) |
| `attack` | 6      | Замах + удар + возврат            |
| `hit`    | 3      | Реакция на удар (откат)           |
| `block`  | 3      | Блок щитом / парирование          |
| `death`  | 6      | Падение                           |

> Спрайт-листы выстраиваются горизонтально: все кадры одной анимации в одну строку.
> Порядок строк фиксируется и одинаков для всех файлов.

## Точки крепления (attachment points)

Позиции в пикселях внутри кадра 64×96. Постоянны во всех кадрах.

```
┌────────────────────┐  (0,0)
│                    │
│      [ HEAD ]      │  ← центр головы: (32, 18)
│                    │
│  [LH]  body  [RH]  │  ← левая рука: (10, 48), правая рука: (54, 48)
│                    │
│      [TORSO]       │  ← центр торса: (32, 52)
│                    │
│      [ LEGS ]      │  ← центр ног: (32, 72)
│                    │
└────────────────────┘  (64, 96)
```

Все оверлеи предметов рисуются относительно этих точек.

## Оверлеи экипировки

Оверлей = спрайт-лист с **теми же размерами кадра и тем же числом кадров**, что и базовый персонаж. Большинство пикселей
прозрачны — видна только сама экипировка.

### Структура файлов

```
assets/
  characters/
    strongman/
      body.png          ← базовая модель Силача

  equipment/            ← оверлеи экипировки
    head/
      none.png          ← прозрачный placeholder
      helmet_iron.png
      hood_leather.png
    hand_right/
      none.png
      sword_iron.png
      dagger.png
      rapier.png
    hand_left/
      none.png
      shield_wood.png
      dagger.png
    body/
      none.png
      chainmail.png
      leather_vest.png
    ring/
      none.png
      ...
    amulet/
      none.png
      ...
    legs/
      none.png
      ...
```

Каталог `assets/equipment/` в репозитории пока не реализован (paper-doll оверлеи — план на будущее, сейчас
у предмета есть только инвентарная иконка `icon.svg`). Структура выше — целевая.

### Порядок слоёв (z-order, снизу вверх)

```
1. body          (базовая модель)
2. legs          (поножи, сапоги)
3. body          (торс, плечи — экипировка слота body)
4. ring          (аксессуар, малозаметный оверлей)
5. amulet        (аксессуар на груди — поверх торса)
6. hand_left     (левая рука / щит — за спиной)
7. head          (шлем, капюшон)
8. hand_right    (правое оружие — поверх всего)
```

## Мобы

Мобы не используют систему оверлеев — у них нет сменяемой экипировки. Каждый моб = один спрайт-лист со всеми анимациями.

| Анимация | Кадров |
|----------|--------|
| `idle`   | 4–6    |
| `attack` | 4–6    |
| `hit`    | 3      |
| `death`  | 6      |

Боссы — отдельные спрайты, могут быть крупнее и иметь больше кадров.

---

## Правило для художника

> Все attachment points должны находиться на одном и том же пикселе во всех кадрах персонажа.
> Это единственное жёсткое требование. Внутри этого ограничения — полная свобода.
