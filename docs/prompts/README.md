# Промпты для генерации фонов и объектов

Генерация — через **Nano Banana** (Gemini image), естественным языком, без флагов Midjourney.

Перед генерацией прочитай [`_style-guide.md`](_style-guide.md) — там правила Nano Banana, палитра, 4-слойная модель
параллакса и конвенция вариантов (по 2 промта на картинку).

## Рекомендуемый порядок генерации

1. Начать с **одной стартовой зоны** (например, Мёртвые поля) — отработать стиль на трёх слоях
2. Показать результат, скорректировать промпт в `_style-guide.md`
3. Сгенерировать остальные две стартовые зоны (Растоптанные луга, Свалка доспехов)
4. Лагерь и карта — после утверждения базового стиля
5. Средние зоны и финальная — в конце

## Соглашение об именах файлов спрайтов

Слой + номер варианта: `<layer>.<n>.png`. После добавления файлов — обнови счётчик в
`src/zones/registry.ts` → `ZONE_BG_VARIANTS`.

```
public/backgrounds/
  camp.png
  map-texture.png
  zones/
    dead-fields/
      far.1.png  far.2.png
      mid.1.png  mid.2.png
      near.1.png near.2.png
      fore.1.png fore.2.png
    mage-ruins/
      far.1.png far.2.png  mid.1.png mid.2.png  near.1.png near.2.png  fore.1.png fore.2.png
    crypt/            far.{1,2}.png  mid.{1,2}.png  near.{1,2}.png  fore.{1,2}.png
    trampled-meadows/ far.{1,2}.png  mid.{1,2}.png  near.{1,2}.png  fore.{1,2}.png
    beast-lair/       far.{1,2}.png  mid.{1,2}.png  near.{1,2}.png  fore.{1,2}.png
```
