# Растоптанные луга ★☆☆

**Фракция:** Конница — Дикие звери
**Тип:** стартовая
**Открывается:** с начала игры

## Лор

Поле где конница делала разгон. Земля взрыта копытами, повсюду сломанные копья и упряжь. Запах крови и гниющей плоти привлёк полчища крыс.

## Мобы

Стартовая зона зверей — **мягкое введение доминанты (число)**: крысы приходят не поодиночке, а сразу выводком, чтобы
игрок с первого боя привык, что тел на доске больше одного. Это семя механики спавна, которую средние зоны
(«Логово зверей») развернут в апекс.

### Крыса `common`

Быстрая, мелкая, почти не опасна поодиночке — но одна никогда не приходит. Кусает и отбегает, наваливаясь числом.
Паттерн: **стартовый выводок** — первое знакомство с числом.

**Механика: призыв** (`summons`, `trigger: start`) — в начале боя с крысой рядом встаёт ещё одна (2 тела на доске).
Призванная крыса сама уже не плодится — стартовый призыв разворачивается только у корневой.

⚠ **HP ниже тира «вводный» (25) сознательно.** Свора — два тела, которых герой убивает **последовательно**, и
второе продолжает атаковать всё время, пока герой добивает первое — суммарная экспозиция вдвое дольше, чем у
одиночного моба той же живучести. Чтобы общая опасность связки не выходила за рамки одного вводного моба, каждое
тело держит `health: 8` — подобрано эмпирически через `npm run balance` (см. ниже), а не по формуле тиров.

- health: 8 / attack_damage: 1 / attack_interval: 1.5s
- Атака: одна полоска, один укус
- Призыв (start): 1× крыса

### Огромная крыса `uncommon`

Разожравшаяся на трупах. Медленнее обычной, но живучее и кусает больнее.

⚠ HP тоже ниже формульного «обычного» тира (50) — тем же эмпирическим соображением: зона в среднем ощутимо
опаснее одиночных зон-соседей стартовой тройки из-за свор, поэтому обе цифры зоны понижены синхронно.

- health: 25 / attack_damage: 1 / attack_interval: 2.5s
- Атака: одна полоска, один удар

### Крысиный король `uncommon` — босс

Чудовищный гибрид: несколько крыс, сросшихся хвостами. Каждая голова атакует сама по себе.

**Механика: мультиатака** — у босса **3 независимых таймера атаки**, по одному на каждую голову. Каждая голова наносит малый урон, но все три атакуют независимо. Убить можно только уничтожив весь общий пул HP.

- health: 100 (общий) / attack_damage: 1 / 1 / 1 (по головам) / attack_interval: 2.0s (на голову)
