# Поле Битвы ★★★

**Фракция:** все три
**Тип:** endless (см. `docs/content.zones.format.md`, `endless.curse_per_fight`)
**Открывается:** после прохождения всех средних зон

## Лор

Эпицентр столкновения трёх армий. Каждый командир, павший в своей зоне, поднимается здесь снова — поле не
отпускает, пока не падёшь ты. Экспедиция не «выигрывается»: она идёт до смерти или ручного отступления
(«В лагерь»), а с каждым пройденным боем растёт проклятие поля — весь входящий по герою урон увеличивается на
`curse_per_fight`% сильнее. Цель — рекордная: продержаться дольше, чем в прошлый раз (см. «Рекорд» в тултипе
зоны на карте и на экране смерти).

## Мобы

Никакого одного финального босса с фазами — рядовые здесь **это все легендарные командиры**, которых игрок уже
побеждал по зонам трёх фракций, собранные в один взвешенный пул (`mob_pool`, `src/zones/battlefield/config.json`).
Каждый бьётся как обычный рядовой моб (без `isBoss`-рамки/крупного спрайта) — своей собственной механикой
(мультиатака/призыв/фаза), уже знакомой игроку по своей исходной зоне:

| Моб (`mob_id`)   | Название           | Механика                                  |
|------------------|--------------------|---------------------------------------------|
| `rat_king`       | Крысиный король    | Мультиатака (3 полоски) + призыв крыс        |
| `dump_boss`      | Главарь свалки     | Флэт-броня                                   |
| `bone_summoner`  | Призыватель костей | Стартовый призыв (2× рунный скелет)          |
| `embalmer`       | Бальзамировщик     | Фаза (→ `swaddled_horror`)                   |
| `pack_leader`    | Вожак стаи         | Стартовый + периодический призыв волков      |
| `herd_stallion`  | Вожак табуна       | Мультиатака (3 полоски) + призыв лошадей     |
| `brigadier`      | Бригадир           | Шипы + фаза (→ `brigadier_light`)            |
| `camp_commander` | Командир лагеря    | Тяжёлая броня + фаза (→ `camp_commander_light`) |

Пул рассчитан на бесконечный набор: мобы выпадают с повторами (взвешенный рандом), длина забега не ограничена.
Лут — обычный `mob_loot` (золото + предметы), падает с каждого убитого моба, как в любой другой зоне; отдельной
награды-выбора в конце нет, потому что конца нет.

## Проклятие

`endless.curse_per_fight` (сейчас 8%, черновое значение до отдельного баланс-прохода — см.
`docs/roadmap.balance.md`) — за каждый пройденный бой множитель входящего по герою урона растёт на это число
процентных пунктов, аддитивно: после N боёв множитель = `1 + curse_per_fight × N / 100`. Сбрасывается только на
новый заход в зону. Текущий уровень проклятия виден под кнопкой «В лагерь» во время экспедиции (тултип
поясняет механику).
